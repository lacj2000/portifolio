# Portifolio

Projeto React (Next.js) com estrutura `src/app/`, `src/app/api/` e `public/`.

Tailwind CSS foi instalado e configurado.

Como usar:

```bash
npm install
npm run dev
```

Abra: http://localhost:3000

Exemplo: a página inicial `src/app/page.jsx` já usa classes do Tailwind para demonstração.

Como rodar:

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000
